# Phantoms 群消息 TTS 自动播报 - v1.0.0.beta

## 📋 项目简介
这是一个用于实时监听并语音播报群聊消息的 Python 工具。
支持增量加载，界面无闪烁，严格逐条播放，支持自定义音色、语速和音量。

## 📋 功能特性
1. ✅ **逐条播报**：严格按顺序播放，播完一条再播下一条。
2. ✅ **增量更新**：只加载新消息，不刷新旧列表，界面无闪烁。
3. ✅ **多音色支持**：自动读取系统所有语音包，自由切换。
4. ✅ **参数可调**：支持语速、音量调节，配置自动保存。
5. ✅ **跳过功能**：可立即停止当前播报并清空队列。
6. ✅ **详细日志**：记录所有操作步骤，方便排错。
7. ✅ **自定义图标**：支持同目录下 `icon.jpg` 作为程序图标。

- Python 3.7 或更高版本
- 依赖库见 `requirements.txt`
